# PdcaCycler
## About
* PDCAを片手で管理できるアプリ
* Plan(計画)-Do(実施/実行)-Check(点検/評価)-Act(処理/改善)のサイクルをワンストップでサポート
* 絶え間ない改善こそ一番の近道

## 流れ
* 1.目標/ゴール設定をする
* 2.目標に紐づく計画をたてる
* 3.計画の実行予定日が過ぎたら振り返る
* 4.改善点をもとに目標を立て直す(3に戻る)

## Features
* 計画や振り返りはログとして残り消えることはない
* どんどん積み重ねていく
* いつでも今までの過程を振り返ることができる
* 終了予定日や期限はリマインダーで知らせてくれる
